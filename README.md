# MU'AWIYE — Portfolio Website

This repository contains a simple personal portfolio website for MU'AWIYE.

Pages included:

- [index.html](index.html)
- [about.html](about.html)
- [skills.html](skills.html)
- [portfolio.html](portfolio.html)
- [services.html](services.html)
- [contact.html](contact.html)

Styles are in `stylesheet.css` and images are in the `web-images/` folder.

To preview the site locally, open `index.html` in your browser or use a static file server.

Example (Python 3):

```bash
python -m http.server 8000
# then open http://localhost:8000
```
